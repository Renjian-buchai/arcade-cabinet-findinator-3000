# Arcade Cabinet Finder-inator 3000

Arcade cabinets are hard to find. The Arcade cabinet finder-inator 3000 solves this by cataloguing cabinets.

## Installation

1. Install dependencies.
   `pip install -r requirements.txt`

2. Run the code.
   `python main.py`
